# Copilot Runtime

The Copilot Runtime is the back-end component of CopilotKit, handling the communication between frontend and server for the message history, state and more.

- https://docs.copilotkit.ai/direct-to-llm/guides/self-hosting
- https://docs.copilotkit.ai/direct-to-llm/guides/self-hosting?endpoint-type=Node.js+Express

### Prerequisites

- Node.js 18+ (recommended: 20+)
- npm or pnpm

### Installation

```bash
npm install
```

### Development

```bash
# Run in watch mode (auto-restarts on file changes)
npm run dev

# Run once
npm start
```

The server will start on http://localhost:4000/api/copilotkit
